# ac_applock

App lock project with native biometric gate.

## Guard Logo Setup

1. Save your logo image as:
   `assets/images/guard_logo.png`
2. Fetch packages:
   `flutter pub get`
3. Generate Android launcher icon from that image:
   `dart run flutter_launcher_icons`

The app UI already displays this guard logo in the home header and setup screen.

## Getting Started
- `flutter run`
